## 1. 对象的实例化

![](https://iqqcode-blog.oss-cn-beijing.aliyuncs.com/img/20200706153033.png)

创建对象`new`的操作并不是原子性的：

```java
Object obj = new Object();
```

对应的字节码指令：

```java
0 new #2 <java/lang/Object>
3 dup
4 invokespecial #1 <java/lang/Object.<init>>
7 astore_1
```

> 这里，联想到了volatile关键字和DCL单例
